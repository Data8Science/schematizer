About


What is this tool?

A browser for documentation on tables in the Redshift Data Warehouse owned by BAM.


Where can I ask questions about a table?

You can directly email the owner of the table, which is listed under "Table Info".


What are categories?

A category is a way for us to aggregate and filter tables. It's entirely user-generated.


How are descriptions and notes different?

The description on a table is pulled from an engineer's source code and cannot be edited.
Notes, on the other hand, are user-generated content -- meant to surface gotchas, joins, and other useful knowledge.


How often is the reference updated?

You can check when a specific table was updated by the "Last Updated" column under Table Info.
